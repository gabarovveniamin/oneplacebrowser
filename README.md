# Comet Browser

Мощный браузер на Python с современной архитектурой для масштабирования.

## Особенности

- ✨ Встроенный Chromium WebEngine (PyQt6)
- 📱 Система табов
- 🔖 Закладки и история просмотра
- 🔐 Авторизация через Google OAuth
- 👤 Управление учетными записями
- 💾 SQLite база данных с синхронизацией
- 🎨 Темы оформления (light/dark)
- 🔒 Управление куками

## Требования

- Python 3.8+
- PyQt6
- PyQt6-WebEngine
- Google OAuth (опционально)

## Установка

```bash
# Установка зависимостей
pip install -r requirements.txt

# Или через setup.py
python setup.py install
```

## Запуск

```bash
python -m src.main
```

## Новые функции

### 🔐 Google OAuth Авторизация
- Вход через учетную запись Google
- Безопасное хранение токенов
- Режим гостя для приватного просмотра

### 📊 История просмотра
- Автоматическое сохранение всех посещенных страниц
- Поиск по URL и названиям
- Очистка истории
- Просмотр времени посещения каждой страницы

### 🌟 Улучшенные закладки  
- Ссылка дружелюбное сохранение
- Быстрый доступ к закладкам
- Удаление ненужных закладок

Подробнее in [GOOGLE_OAUTH_SETUP.md](GOOGLE_OAUTH_SETUP.md)

## Использование

### Базовые функции

- **Переход по URL**: Введите URL в адресную строку
- **Поиск**: Введите текст без протокола - автоматически будет выполнен поиск Google
- **История**: View → History
- **Закладки**: Нажмите ★ чтобы добавить закладку

### Горячие клавиши

- `Backspace` - Назад
- `Shift+Backspace` - Вперед
- `F5` - Обновить
- `Ctrl+T` - Новая вкладка

## Архитектура

### Модульная структура

Проект разделен на независимые модули:

- **browser** - Основной движок браузера
- **networking** - Сетевые запросы и кэширование
- **storage** - Персистентное хранилище данных
- **ui** - Пользовательский интерфейс
- **utils** - Вспомогательные функции

### Масштабирование

Архитектура позволяет легко расширять функциональность:

1. Добавить новый модуль в соответствующую папку
2. Экспортировать через `__init__.py`
3. Использовать в главном окне или других модулях

Пример:
```python
from src.networking import HttpClient
from src.storage import DatabaseManager

# Использовать...
```
